1. Overview of the Service

Voilo is a productivity application that helps users record voice input and convert it into structured data that can be written into Google Sheets. The App performs the following:

  • Converts speech to text using Apple's Speech API
  • Extracts structured data using OpenAI APIs
  • Writes data into your own Google Sheets through the Google Sheets API
  • Stores tracker configurations and pending entries locally on your device using Apple SwiftData

Voilo does not:

  • Store your voice recordings after they are processed
  • Maintain servers that store personal data
  • Access or read any Google Drive files except those you explicitly select or create for use with Voilo

Future versions may use a backend server for anonymized click-stream analytics to improve usability. Personal data will not be stored without user disclosure and consent.
